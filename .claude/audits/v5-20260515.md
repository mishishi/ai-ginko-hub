## P1 — 高优问题 (HIGH): 2 项

### P1-1: 搜索框缺少 `inputmode="search"`

**文件**: `src/components/Header.tsx:37`
**参考**: UI/UX Pro Max UX — "Mobile Keyboards: Show appropriate keyboard for input type"

**问题**: 搜索 input type="text" 在移动端会调起标准文字键盘，而非搜索专用键盘（带搜索按钮）。

**当前**:
```tsx
<input id="header-search" type="text" ... />
```

**修复**:
```tsx
<input id="header-search" type="search" inputMode="search" ... />
```

---

### P1-2: 筛选结果数量变化无 ARIA live 区域

**文件**: `src/App.tsx:103-106`
**参考**: UI/UX Pro Max UX — "Aria Live: Async updates need aria-live for screen readers"

**问题**: 当用户筛选标签或搜索时，项目数量变化（"找到 N 个结果"）不会向屏幕阅读器播报。

**当前**:
```tsx
<span className="text-sm text-text-muted">
  找到 {filtered.length} 个结果
</span>
```

**修复**:
```tsx
<span className="text-sm text-text-muted" aria-live="polite" aria-atomic="true">
  找到 {filtered.length} 个结果
</span>
```

## P2 — 建议优化 (MEDIUM/LOW): 4 项

### P2-1: 精选标签 `.visible_&` 选择器过于复杂

**文件**: `src/components/ProjectCard.tsx:99`
**问题**: `[.visible_&]:opacity-100 [.visible_&]:delay-[300ms]` 是 Tailwind 4 arbitrary variant 语法，依赖 `.visible` 父选择器生效。如果父级变化（添加其他 class），选择器可能失效。

**建议**: 改用 CSS 动画 `animation-fill-mode: forwards` 和 `animation-delay`，在 `.reveal.visible` 中控制精选标签的动画。

---

### P2-2: 项目卡片 role 条件切换可简化

**文件**: `src/components/ProjectCard.tsx:73`

**当前**:
```tsx
role={isPlaceholder ? 'article' : 'button'}
tabIndex={isPlaceholder ? -1 : 0}
```

**观察**: 占位符时 `role="article"` 但仍有 `onClick`（空操作）和 `onKeyDown`。功能正确但语义略有歧义。

**建议**: 统一用 `role="article"`，`onClick` / `onKeyDown` 在 `isPlaceholder` 时直接不绑定（或条件渲染外层 div）。

---

### P2-3: 未使用的 `label-pulse` 动画

**文件**: `src/index.css:171-175`

**问题**: 定义了 `@keyframes label-pulse` 但无任何元素引用，遗留代码。

**建议**: 移除。

---

### P2-4: Google Fonts 预连接可添加 `dns-prefetch`

**文件**: `index.html:20-21`

**当前**:
```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
```

**建议**: 添加 `dns-prefetch` 作为 preconnect 的降级：
```html
<link rel="dns-prefetch" href="https://fonts.googleapis.com" />
<link rel="dns-prefetch" href="https://fonts.gstatic.com" />
```
